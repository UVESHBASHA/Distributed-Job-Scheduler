# Distributed Job Scheduler

A production-inspired distributed job scheduling platform built using FastAPI, PostgreSQL, React, and Python Workers.

## Features

- Authentication
- Organizations
- Projects
- Job Queues
- Distributed Workers
- Retry Policies
- Dead Letter Queue
- Dashboard
- Metrics
- Live Monitoring

## Tech Stack

### Backend
- FastAPI
- SQLAlchemy
- PostgreSQL
- Alembic

### Frontend
- React
- TypeScript
- Tailwind CSS

### Worker
- Python
- AsyncIO
